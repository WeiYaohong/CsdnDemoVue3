# 版本

Node 14.19.1

# 开始使用

yarn install
yarn serve

# 说明

1、纯前端项目，未配置后端接口，数据为模拟数据
2、菜单路由配置地址 src/router/index.js
3、项目基础配置地址 src/config/setting.config.js
4、主题相关配置 src/config/theme.config.js,主题样式 library/styles
